# 🎨 むしマップ デザインショーケース

## 📱 **UIデザインの特徴**

### 🌈 **カラーパレット**
```
🍃 プライマリ: #4CAF50 → #2E7D32 (自然の緑グラデーション)
🔥 アクセント: #FF6B6B → #FF8E53 (ライブ配信の暖色)
💙 AI機能: #2196F3 → #1976D2 (知的な青グラデーション)
🌸 ソフト: #E3F2FD → #BBDEFB (優しいアクセント)
```

### ✨ **アニメーション効果**
- **フェードイン**: 0→1 (800ms)
- **スライドアップ**: 50px→0 (800ms) 
- **プログレスバー**: 0→100% (2000ms)
- **レベルアップ**: スケール + 回転エフェクト

### 🎭 **影・立体効果**
```css
elevation: 6
shadowColor: #4CAF50
shadowOffset: { width: 0, height: 4 }
shadowOpacity: 0.3
shadowRadius: 8
```

- 🎨 グラデーションヘッダー

- 🤖 AI識別ボタン（光るエフェクト）

### 2. **チャット画面 (ChatInterface)**
- 💬 メッセージバブル（自分=緑、他人=グレー）
- 😊 リアクション絵文字システム
- 🟢 オンライン状況インジケーター
- ⌨️ 入力中の「...」アニメーション

### 3. **ライブ配信 (LiveStreamingScreen)**
- 🔴 脈動するLIVEバッジ

- 🎬 映画風のプレーヤーデザイン

### 4. **協力探索 (CollaborativeDiscoveryScreen)**
- 🗺️ 探検風のカードデザイン

- 🔍 特徴検出のハイライト表示
- ✨ 結果発表の劇的演出

## 🎯 **デザインシステム統一要素**

### 📐 **レイアウト原則**
- **余白**: 15-20px統一
- **角丸**: 12-15px（親しみやすさ）
- **カード間隔**: 15-20px
- **ボタン高**: 48px（タッチしやすさ）

### 🎨 **UI要素スタイル**
```
ボタン: 角丸 + グラデーション + 影
カード: 白背景 + 軽い影 + ホバー効果
入力欄: 境界線 + フォーカス時ハイライト
バッジ: 小さな角丸 + テーマカラー
```
